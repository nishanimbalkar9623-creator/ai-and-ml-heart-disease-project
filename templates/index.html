About Project

How Medicare Works

A full-stack academic project combining machine learning, clinical data science, and modern web design to make cardiovascular risk assessment accessible.

Project Overview

Medicare is an AI-powered heart disease prediction system built as an academic project. It uses the UCI Heart Disease dataset (Cleveland) with 303 patient records and 14 attributes to train a Random Forest classifier capable of predicting the presence of heart disease with high accuracy.

The project demonstrates the integration of machine learning models with a modern web frontend and a Python Flask backend REST API — creating a complete clinical decision support system prototype.

Technical Pipeline

1

Data Collection

UCI Heart Disease Cleveland dataset — 303 records, 13 features + target variable (0=no disease, 1=disease).

2

Preprocessing

Handled missing values, feature scaling with StandardScaler, one-hot encoding for categorical variables.

3

Feature Selection

Top features by importance: cp (chest pain), thal, ca (vessels), thalach (max HR), oldpeak (ST depression).

4

Model Training

Random Forest with 100 estimators, max depth 10, trained with 80/20 train-test split + 5-fold cross validation.

5

Deployment

Model serialised with Pickle, served via Flask REST API endpoint /predict, consumed by this frontend.

Overview

Medicare API

The Medicare prediction API is a Flask-based REST service that accepts 13 clinical parameters and returns a binary prediction (0 = no disease, 1 = disease present).

⚠ This API is for academic/demo purposes only and must not be used for real clinical decisions.

Base URL

http://127.0.0.1:5000

Quick example

const response = await fetch('http://127.0.0.1:5000/predict', {
method: 'POST',
headers: { 'Content-Type': 'application/json' },
body: JSON.stringify({ age:54, sex:1, cp:0, trestbps:130, ... })
});
const { result } = await response.json();
// result: 0 (no disease) or 1 (disease present)

app.py structure

from flask import Flask, request, jsonify
from flask_cors import CORS
import pickle, numpy as np

app = Flask(__name__)
CORS(app)
model = pickle.load(open('model.pkl', 'rb'))

@app.route('/predict', methods=['POST'])
def predict():
data = request.json
features = [data[k] for k in FEATURE_KEYS]
pred = model.predict([features])[0]
return jsonify({ 'result': int(pred) })
API

POST /predict

Accepts a JSON body with 13 clinical parameters and returns a prediction.

Request

POST /predict
Content-Type: application/json

{
"age": 54, "sex": 1, "cp": 0, "trestbps": 130,
"chol": 250, "fbs": 0, "restecg": 0, "thalach": 150,
"exang": 0, "oldpeak": 1.5, "slope": 1,
"ca": 0, "thal": 2

{ "result": 0 } // 0 = no disease, 1 = disease present

Error response

{ "error": "Missing required field: age" }
